Microsoft To Do CLI

This project is not affiliated or associated with Microsoft. This application only uses the Microsoft Graph API .

Who am I?

I'm Lander Van laer, a student of Applied Artificial Intelligence at the De Nayer Campus of Thomas More

What?

The Microsoft To Do CLI is an unofficial CLI-application for the Microsoft To Do. Where it is possible to create tasks, delete tasks, move tasks between task lists and more...

It is also possible to use the application on a server, so that as soon as an error occurs, a new task will be made.
